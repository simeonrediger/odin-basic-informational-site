# Odin Basic Informational Site

A simple Express web server that serves a few pages.

## Setup

1. Ensure you have the latest versions of Node.js and npm installed.
2. Clone the repo to your machine.
3. `cd` into the repo directory and run the following:

```bash
npm install
cp .env.example .env
```

## Run the server

```bash
npm start
```
